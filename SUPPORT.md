# Support

## How to Get Help

Use the following paths depending on the problem:

- usage question: open a GitHub issue with enough context
- bug report: use the bug report issue template
- feature request: use the feature request issue template
- security concern: follow [SECURITY.md](SECURITY.md) and use a non-public maintainer contact path or GitHub private vulnerability reporting if available

## What to Include

For most support requests, include:

- your macOS version
- whether the videos are on a local drive or external drive
- steps to reproduce the problem
- expected behavior
- actual behavior
- logs or screenshots if relevant

## Maintainer Notes

If this repository gets a dedicated email address, GitHub Discussions board, or
other official support channel later, update this file and the other community
documents to point to that channel.
